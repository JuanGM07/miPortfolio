DataBaseQuery

Esta herramienta es un modelo NL2SQL, la cual recibe lenguaje natural y lo transforma a SQL para comunicarse con una base de datos. Ideal para que empleados que no conozcan de este lenguaje puedan hacer consultas a las bases de datos de la empresa.

Esta base de datos es de prueba, y solo muestro 10 filas para que no se ralentize la carga, como ejemplo.

¿Cómo funciona DataBaseQuery?

1. Base de datos

Para que tenga sentido usar esta herramienta debemos de tener una base de datos. Yo utilizo una de prueba, de e-commerce, si ejecutais el codigo de prueba vereis que solo muestro 10 filas para que se visualice, pero tengo en cuenta la base de datos al completo en la logica de programación.

2. Escribir y enviar la query

Después escribimos nuestra query, que es lo que queremos saber de nuestra base de datos, escrito en lenguaje natural, lo que lo hace muy útil para empleados que no sean tan técnicos, ni conozcan lenguaje SQL.

3. Visualización del Resultado

Finalmente, solo nos queda ver el resultado de la query. Y hacer todas las que necesitemos, solo dependerá del límite fijado por el gasto de nuestra API de OpenAI

Características clave:

  • Conversión de Lenguaje Natural a SQL (NL2SQL)
  • Base de Datos de Prueba Incorporada
  • Preparado para Escalabilidad
  • Interfaz Simple y Accesible
  • Diseño compacto y adaptado a pantallas 1080p

Tecnologías utilizadas:

Python Flask OpenAI GPT Pandas SQlite Langchain
