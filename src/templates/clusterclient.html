ClusterClient

ClusterClient es una herramienta interactiva diseñada para segmentar clientes automáticamente a partir de datos como ingresos anuales y puntuación de gasto.

Utiliza algoritmos de machine learning para identificar grupos de comportamiento similares y proporciona visualizaciones y recomendaciones prácticas para ayudar a las empresas a entender mejor a sus clientes y tomar decisiones estratégicas basadas en datos.

¿Cómo funciona ClusterClient?

1. Carga del archivo CSV

Primero arrastramos o elegimos el archivo CSV que queremos cargar para analizar. Es exclusivamente para archivos CSV, aunque si manejan otro tipo de archivos siempre se puede cambiar la funcionalidad.

2. Análisis de Clusters

A continuación carga automaticamente el análisis de clusters del CSV cargado, tanto una visualización de los clusters, en este caso teniendo en cuenta principalmente el spending score (gasto) y el ingreso anual. Para aplicar diferentes estrategias en función del cluster al que pertenezca el cliente. Debajo aparecen los distintos clusters con sus características y el total de clientes.

3. Descarga del CSV con clusters

Por último podemos descargar el CSV con los clusters al que pertenece cada cliente, además de esto tendremos otros datos de contacto como e-mail o número de teléfono, teniendo esto y sabiendo a que cluster pertenecen podemos implementar una estrategia comercial diferente para cada uno de ellos.

Características clave:

  • Análisis automático de clusters con K-Means
  • Visualización interactiva con Plotly
  • Interpretación de segmentos
  • Subida de archivos CSV mediante drag & drop
  • Diseño compacto y adaptado a pantallas 1080p

Tecnologías utilizadas:

Python Flask Scikit-Learn Pandas Plotly
